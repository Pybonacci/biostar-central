RSS Feeds

Feeds estándar:

  • Últimas entradas - últimas entradas (todas las entradas)
  • Nuevas preguntas - últimas preguntas
  • Nuevas ofertas de trabajo - últimas ofertas
  • Planet Feed - Últimas entradas de blog

Uso avanzado, se pueden añadir múltiples fuentes a un feed.

  • Seguir entradas de diferentes tipos, entradas de determinado tipo: /feeds/type/job+blog
  • Seguir múltiples entradas, respuestas y comentarios anexadas a las entradas específicas: /feeds/post/2+8+10
  • Seguir múltiples etiquetas, entradas etiquetadas con una o más etiquetas: /feeds/tag/galaxy+bwa+tutorial
  • Seguir a múltiples usuarios un feed para esos usuarios: /feeds/user/2+8+10
